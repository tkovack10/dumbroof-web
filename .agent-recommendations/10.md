# Agent Recommendation #10

## Summary
Material tag defaulting to comp_shingle_laminated instead of asphalt_shingle or correct metal subtype (3× this week, severity=medium)

## Rationale
Add material classification guidance: do not default to 'comp_shingle_laminated' unless the dimensional/laminated profile (visible shadow lines, multi-layer tabs) is clearly discernible. For standard 3-tab or indeterminate shingles, use 'asphalt_shingle'. For metal surfaces, distinguish 'metal_roofing' (panels, standing seam, exposed fastener panels) from 'metal_flashing' (drip edge, step flashing, pipe collar).
