## Inspector / Third-Party Patterns

## Effective Counter-Arguments
1. **HOVER data contradiction:** Carrier's own HOVER report contains all the measurements they omitted — ridge/hip, step flashing, wall flashing, gutters, soffit, fascia. Cite specific HOVER numbers against their scope.
2. **House wrap corner rule (RCNYS R703.1/R703.2):** Carrier omitted WRB entirely. Code requires continuous weather-resistant barrier around all corners — forces full elevation siding when any elevation is replaced.
3. **Waste factor:** 5.7% is demonstrably inadequate for complex roof geometry. NRCA guidelines recommend 12% minimum for roofs with multiple valleys and dormers.
4. **Steep + high roof charges:** HOVER explicitly confirms pitch and story count. Both remove AND install charges apply.
5. **O&P justification:** Multi-trade coordination argument — 5 trades (roofing, siding, window capping, gutters, fascia) require GC management.
6. **Siding quantity:** Direct HOVER comparison shows 247 SF shortage — carrier used 1,305 SF vs. HOVER's 1,552 SF siding facades.

## Pricing Patterns
