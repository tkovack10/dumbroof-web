## Known Tactics

### 1. NYBI8X Price List (Below Current NYBI26)
Nationwide uses NYBI8X_JUL25 — systematically lower than current NYBI26 on virtually every line item. Key differences:
- Ridge cap: $7.79/LF vs $11.97 NYBI26 (35% lower)
- Drip edge: $3.87/LF vs $4.21 (8% lower)
- Laminated shingle install: $333.37/SQ vs $337.78 (1.3% lower)
- House wrap: $0.54/SF vs $0.64 (16% lower)
- Vinyl siding: $7.34/SF vs $7.55 (2.8% lower)

### 2. Partial Elevation Siding
1st scope limited siding to 2 partial elevations on dwelling (710 SF of 2,168 SF = 33%) and 60 SF spot repair on garage (6% of 1,071 SF). Same tactic as Assurant — carrier attempts to limit siding to only the most obviously damaged areas rather than full-perimeter replacement required by code.

### 3. Single-Layer Tear-Off on Multi-Layer Roofs
1st scope priced only single-layer tear-off despite confirmed two-layer roof system on both dwelling and garage. Added 2nd layer line in revised scope after supplement.

### 4. No O&P Despite Multi-Trade Scope
1st scope applied $0 O&P to ALL structures despite dwelling having 5+ trades and garage having 3+ trades. Revised scope added O&P to dwelling + garage but still excluded barn (3+ trades).

### 5. Omit Decking Replacement
No plywood decking replacement in either original or revised scope despite 17 sheets (544 SF) replaced during tear-off. Carrier avoids this line item entirely.

### 6. Inadequate Debris Removal
1st scope included only 0.5 truck load ($91.62) for a 28+ SQ two-layer dwelling tear-off. Revised scope added 1 dumpster load ($638.95) — still inadequate for the scope of work.

### 7. Omit Labor & Equipment
No roofer labor hours, equipment operator, or adequate dumpster capacity in any scope.

### 8. Classify Code Upgrades as "Paid When Incurred"
Revised scope classified house wrap and valley I&W as "Ordinance or Law — Paid When Incurred." This separates these items from the main scope and subjects them to a different payment timeline, potentially delaying reimbursement.

## Effective Counter-Arguments

### What Worked (Drove $77K → $122K = +$44K):
1. **House wrap corner rule (RCNYS R703.1/R703.2)** — forced full-perimeter siding on dwelling and garage (from 710 SF → 2,964 SF)
2. **Two-layer tear-off code requirement (RCNYS R908.3.1.1(3))** — forced 2nd layer line items on dwelling + garage
3. **Valley I&W code requirement (RCNYS R905.1.2)** — added 328 SF valley I&W
4. **Photo documentation of no house wrap** — smoking gun evidence that fanfold ≠ weather-resistant barrier
5. **Full gutter/downspout replacement** — expanded from partial to comprehensive system
6. **O&P multi-trade argument** — carrier conceded O&P on dwelling + garage
