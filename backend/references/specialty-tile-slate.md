# Specialty Tile & Slate Claims — Tactics Deep Dive

> Source: Precision Construction & Roofing claims intelligence, adapted for DumbRoof.AI platform.
> These claims are the highest-value, highest-friction claims in residential roofing.
> Average claim value: $35,000–$200,000+ (vs. $8,000–$15,000 for standard shingle).

---

## The C.A.R. Framework — Compatibility, Availability, Repairability

Every specialty tile/slate claim runs through three gates in sequence. If ANY gate fails, the claim trajectory changes.

### Gate 1: Compatibility
**Question:** Can the replacement tile physically interlock with and match the existing system?

**What to Document:**
- Tile manufacturer, product line, and profile designation (e.g., "Ludowici Nantucket Flat" vs "Ludowici New England Traditional")
- Physical measurement of the tile profile — length, width, head lap, sidelap dimensions
- Interlocking mechanism: pan-and-cover, S-tile, flat interlocking, or double-roman profile
- Side-by-side comparison photos of existing tile vs. available replacement tile
- **Written statement from manufacturer's representative on company letterhead confirming incompatibility**
- Request building code official's written determination if incompatibility makes partial repair impossible under code

> A written letter from Ludowici's technical department saying "Profile X cannot be combined with Profile Y in the same roofing plane" is an O&L trigger.

### Gate 2: Availability
**Question:** Is the matching tile currently in production and available for purchase?

| Status | Claim Impact | Required Documentation |
|--------|-------------|----------------------|
| Currently Available | Repair is possible in scope | Manufacturer catalog listing + lead time quote |
| Long Lead Time (12-18 mo) | Additional coverage for temp protection, storage, mobilization | Written lead time confirmation from manufacturer |
| Temporarily Unavailable | Treat as discontinued if >90 days | Written status letter from manufacturer |
| Permanently Discontinued | Full replacement required — **strongest O&L trigger** | Written discontinuation letter on manufacturer letterhead |

**Ludowici-specific:** Three status designations: "Available," "Temporarily Unavailable," and "Permanently Discontinued." Only the first allows repair-in-kind.

### Gate 3: Repairability
**Question:** Can the damaged sections be repaired without requiring full system replacement?

**What Blocks Repairability:**
- Substrate damage (deck rot, sheeting failure) beneath tile requiring removal of adjacent undamaged tile
- Copper valley or flashing systems that cannot be patched without full replacement
- Mortar-set ridge systems that cannot be selectively repointed without disturbing anchored tiles
- Lead-coated copper flashings that are discontinued and cannot be sourced to match
- Code violations uncovered during repair requiring >25% of roof section to be brought into compliance

### C.A.R. Decision Tree

```
Compatibility FAILS → Full replacement, O&L applies
Availability FAILS (discontinued) → Full replacement, O&L applies
Availability FAILS (long lead) → Repair/replacement with extended coverage for delay costs
Repairability FAILS → Full replacement, O&L may apply if >25% of roof section
All three PASS → Targeted repair with complete supplement list
```

## Ordinance & Law (O&L) Coverage — The Complete Framework

O&L (Code Upgrade Coverage) is the single highest-value add-on in specialty roofing claims. A single O&L trigger on a $60,000 clay tile replacement with 25% O&L coverage = **$15,000 in additional coverage**.

### The Three Parts of O&L Coverage

| Part | What It Pays For | Example |
|------|-----------------|---------|
| **Coverage A: Undamaged Portion** | Tear out and replace undamaged parts that must be demolished to comply with current code | Tile in slopes adjacent to damaged slope that must be removed because 25% rule requires full replacement |
| **Coverage B: Demolition Costs** | Demolition and debris disposal for undamaged sections that must be removed | Labor + disposal for removing 18 SQ of undamaged north slope tile |
| **Coverage C: Increased Cost of Construction** | Premium between standard and code-compliant materials/methods | Impact-resistant underlayment, upgraded fastening, I&W shield, radiant barrier required by current IRC but not original spec |

All three can be triggered simultaneously and the carrier owes each part independently. Coverage amounts typically offered at 10%, 25%, or 50% of dwelling coverage.

### O&L Trigger 1: The 25% Rule (Most Common)

**Code:** IBC Section 706.8.1 / IRC Section R908.4

**Rule:** If the area of repair or replacement in any 12-month period exceeds 25% of a roof **section**, the entire roof section must be brought into full compliance with current code.

**Critical:** Calculate by SECTION (front slope, rear slope, dormers), NOT whole roof. On a hip roof with four equal slopes, 7 damaged squares out of 28 total is 25% of the whole roof — but if concentrated on ONE slope of 14 squares, that's 50% of that section and unambiguously triggers O&L.

**Documentation:**
- Calculate total roof area by section — NOT the entire roof
- Document damage within each section independently using test squares
- Xactimate area measurements with section labels; diagram showing damaged vs. total section area
- Attach applicable IBC/IRC code section verbatim as F9 note or supplemental document
- Obtain written permit condition from local building department confirming full replacement requirement

### O&L Trigger 2: Material Incompatibility + Code Official Determination

When a discontinued or incompatible tile profile cannot be physically matched, and a local building code official provides a written determination that partial replacement is code-prohibited, O&L is triggered. This is the **Schnell doctrine** (see Legal Cases below).

**Documentation:**
- Manufacturer's written incompatibility or discontinuation letter
- Local building department written determination: "Partial replacement of [Profile] tile in combination with [Replacement Profile] tile is not permitted under [Code Section] because the profiles are incompatible and cannot form a weather-tight assembly"
- If carrier denies O&L based on the letter, cite **Schnell v. State Farm Lloyds (5th Cir. 2024)**

**Carrier tactic:** Carriers argue the code official's determination was "conditional" or "advisory." Schnell was remanded precisely on this — the court held that whether the determination was absolute vs. conditional is a genuine issue of material fact for trial.

### O&L Trigger 3: Code Upgrade Requirements During Permitted Work

When a roofing permit is pulled for covered storm damage repair, the building inspector requires all replaced work meets current code. Older homes commonly fail:

| Code Upgrade Item | IRC Reference | Typical Cost Impact |
|-------------------|--------------|-------------------|
| Ice & water shield | IRC R905.2.7.1 | $800–$3,500 |
| Enhanced fastening pattern | IRC R905.3.4 / ASCE 7 wind zone | $1,200–$4,000 |
| Drip edge (all eaves/rakes) | IRC R905.2.8.5 | $600–$2,200 |
| Ventilation upgrade | IRC R806.2 | $1,500–$6,000 |
| Deck upgrade (Exposure 1) | IRC R803.1 | $2,000–$12,000 |
| Impact-resistant materials | Local ordinance / IRC R905.2.4 | $3,000–$15,000 |

Each upgrade cost is a Coverage C item under O&L. Document each with specific code section and line-item cost comparison between old standard and new requirement.

### O&L Documentation Package (Submit Every Time)

- Separate O&L coverage demand letter (NOT buried in Xactimate notes)
- Manufacturer compatibility or discontinuation letter
- Building department written determination or permit conditions
- Code sections cited verbatim — IBC/IRC section and subsection
- Xactimate estimate with O&L items on separate line item group (Coverage A / B / C segregated)
- Before/after cost comparison showing code upgrade premium
- Any applicable legal citations (Schnell, state appellate decisions)

## Hail Damage Documentation — Tile & Slate

Tile does not bruise. It fractures, chips, cracks, or spalls. Documentation must prove: (1) damage is functional, not cosmetic; (2) damage is fresh and storm-related; (3) damage density per test square meets replacement threshold.

### Functional vs. Cosmetic Damage

**Functional Damage (InterNACHI Standard):** Damage that compromises the roof's ability to shed water, reduces expected service life, or structurally undermines the roofing assembly.

| Damage Type | Classification | Why |
|-------------|---------------|-----|
| Corner fractures exposing raw substrate | FUNCTIONAL | Water entry point |
| Through-cracks spanning full width/length | FUNCTIONAL | Breaks water shed plane |
| Spalls removing >10% of tile face area | FUNCTIONAL | Reduces service life below threshold |
| Impact craters penetrating tile glaze | FUNCTIONAL | Water absorption begins at exposed porous substrate |
| Hairline surface cracks, no penetration | MAY BE COSMETIC | Requires engineering assessment |
| Surface crazing / glaze fractures without depth | LIKELY COSMETIC | Unless water infiltration demonstrated |

> Use a moisture meter on impact areas during inspection. If a fracture has allowed water penetration, the moisture reading is proof of functional damage regardless of how "minor" the fracture appears. Document meter model and calibration date.

### Hail Size Thresholds for Tile Damage

| Tile Type | Min Hail Size for Functional Damage | Full Breakage Size |
|-----------|--------------------------------------|-------------------|
| Clay/ceramic tile | 1.25" (quarter-size) | 1.5" (ping-pong ball) |
| Concrete tile | 1.5"–2.0" | 2.0"+ |
| Slate (hard) | 1.75"–2.5" | 2.5"+ |
| Slate (soft/weathered) | 0.75"–1.25" | 1.25"+ |
| Terracotta barrel tile | 1.0"–1.5" | 1.5"+ |

**Carrier tactic:** "Hail smaller than threshold cannot cause observed damage."
**Counter:** (1) Multiple simultaneous impacts weaken tile before critical hit; (2) hail velocity varies from reported size; (3) angle of impact is never perpendicular to slope; (4) existing weathering and thermal fatigue reduce threshold.

### Test Square Methodology

**Protocol:**
1. Mark each 10' x 10' test square with chalk or paint stick — photograph the marking
2. Number each square and create diagram showing locations on roof plan
3. Within each square, photograph and number every impact individually
4. Classify each hit: functional fracture, corner break, surface spall, or cosmetic
5. Record impact count by type for each square on standardized form
6. Count only FUNCTIONAL hits toward replacement threshold
7. Take overview photo of entire marked square showing all damage

**Replacement Threshold:** 7–10 functional impacts per 10' x 10' test square triggers full section replacement recommendation on tile. Document hit density for EVERY square — do not cherry-pick.

### Fresh vs. Pre-Existing Fracture Identification

| Characteristic | Fresh (Storm) Fracture | Pre-Existing Fracture |
|---------------|----------------------|---------------------|
| Fracture face | Bright, unweathered aggregate — same color as tile body | Darkened, stained, or soiled aggregate |
| Edge sharpness | Sharp, angular edges with no rounding | Rounded or worn edges |
| Dirt/debris | None or minimal inside fracture | Dirt, moss, lichen, or oxidation visible |
| Matching fragment | Fragment usually present or traceable | Fragment missing, weathered, or fused |
| Substrate color | Exposed substrate matches body color | Substrate darkened/stained from infiltration |
| Efflorescence | Absent | White mineral deposits around old cracks |

> Always inspect the attic below damage areas for fresh water staining or active moisture. A fresh water stain directly below a fracture photographed the same day is nearly impossible for carriers to dismiss as pre-existing.

## Engineer Report Battle — Build Yours, Counter Theirs

### Common Deficiencies in Carrier Engineer Reports

| Red Flag | How to Counter |
|----------|---------------|
| Wrong material identification | Compare to product stamps photographed during inspection; formal objection citing material misidentification |
| No hail velocity calculations | Request mathematical basis in writing; cite AMS hail testing research for the specific material |
| Generic boilerplate language | FOIA/subpoena for other reports by same engineer; document template language patterns |
| Cosmetic-only finding without testing | Counter with moisture meter readings, bore scope attic inspection, engineer's own ASTM standard |
| No functional damage standard cited | Your report cites ASTM/TRIA/InterNACHI; their report doesn't — credibility gap |
| Unrelated engineering background | Challenge qualifications in writing; request CV and roofing-specific projects; file licensing board complaint if materially false |

### Building the Contractor-Side Engineering Assessment

**Elements of a Credible Report:**
1. **Inspector Qualifications** — PE with documented roofing experience or TRIA inspector certification
2. **Scope and Methodology** — Test squares, number of tiles inspected, equipment used, time on site
3. **Material Identification** — Manufacturer, product line, profile, production era, current status
4. **Damage Classification** — Each impact classified with reference to applicable standard (TRIA Technical Brief, ASTM C406, InterNACHI SOP 5.4)
5. **Fracture Analysis** — Fresh vs. pre-existing determination with physical basis
6. **Functional Impact Density** — Count by test square with replacement threshold analysis
7. **Code Analysis** — 25% rule applicability, code upgrades required under permitted repair
8. **Conclusion** — Explicit statement of whether full section replacement is warranted and why

> NEVER submit a contractor's own written "engineering report" without PE licensure. Inspection report establishes scope. Engineer report establishes the technical and code basis for replacement. Conflating them weakens both.

## Supplement Checklist — Universal Tile Claim Line Items

These items are owed on virtually every tile roof insurance claim. Add them to the initial estimate — don't wait to supplement.

| Line Item | Xactimate Code | Why It's Owed | Carrier Objection & Counter |
|-----------|---------------|--------------|---------------------------|
| Starter Course | RFC STARTER | Every re-roof requires new starter — cannot reuse | "Can be reused" → Starter is nailed through by field tiles; reuse voids manufacturer warranty |
| Ridge & Hip Cap | RFC RDG | Separate labor rate; factory-made units required | "Same rate as field tile" → Xactimate has separate code; factory caps have different pricing |
| Step Flashing | RFG SFL | Cannot reuse — nailed through by tile courses | "Can be reused" → Tile nailed through step flashing; removal tears through existing |
| Head Wall / End Wall Flashing | RFG HWFL | Flashing at wall terminations must be replaced | Carrier omits entirely → Code requires new flashing when tile assembly replaced |
| Valley Lining | RFC VALM | Metal valley nailed through; cannot reuse same holes | "Can be reused" → New tile through same holes creates leak points |
| Paint / Coat Flashings & Vents | PNT MTLR | Galvanized metal oxidizes; painting required | Carrier omits → Code + warranty require painted metal to prevent galvanic reaction |
| Drip Edge | RFG DE | IRC R905.2.8.5; often missing on older homes | "Existing can stay" → Felt/I&W must go under at eave, over at rake; removal required |
| Gable Cornice Returns | CRP GBL | Hip and gable trim at rakes must be replaced | "Not damaged" → Required for proper tile termination and warranty compliance |
| Ice & Water Shield | RFG IWS | Current IRC mandates at eaves, valleys, penetrations | "Not in original specs" → Coverage C O&L item — current code applies |
| Deck Repair/Replace | RFG DKR | Hidden until tile removal | "Not in original scope" → Cannot be known until tile removed; supplement with photos |
| O&P (10/10) | OVERHEAD | GC coordination when 3+ trades | "Direct hire" → Three-trades rule applies |
| Dumpster / Debris | CLN DUMP | Tile weighs 900–1,100 lbs/SQ — multiple dumpsters | "Standard debris" → Weight documentation: squares x weight per square |
| Permit & Inspection | GNL PRMT | Permit required for structural re-roof | Carrier omits → Legally required; provide actual fee from jurisdiction |
| Scaffold / Lift | SCF SCAF | Safety compliance on multi-story or steep-slope | "Ladder access" → OSHA fall protection; steep-slope access by lift is standard |
| Lead Time / Storage / Temp Protection | GNL STOR | Custom Ludowici 12–18 month lead; temp roof needed | "Not covered" → Temp protection is direct result of covered loss; lead time documented |

### Specialty Tile-Specific Additional Items
- **Copper flashing fabrication** — Custom bent copper valleys/counterflashing: $80–$180/LF; never accept standard metal pricing
- **Mortar pointing/repointing** — Ridge, hip, and bedding mortar: separate trade skill with distinct labor rates
- **Tile sorting and salvage** — If discontinued, salvaging undamaged tile from non-damaged sections is billable labor
- **Manufacturer representative inspection** — Ludowici Elite Crown includes factory rep inspection on large projects
- **Historic preservation documentation** — National Register properties require written architectural consultation
- **Hail report / HailTrace** — Professional weather verification report is a direct claim cost

### F9 Note Formula

```
[Line item name] is required because [specific factual reason — not opinion].
Per [code section / manufacturer specification / industry standard], [what is required].
This item is not reusable because [specific reason].
Supporting documentation: [attached document reference].
```

> Write F9 notes as if an appraiser or judge will read them — on contested claims, they will.

## O&P Arguments — Specialty Claims

**Legal Standard:** O&P is owed when a general contractor is "reasonably likely" to be needed — not only when one has been hired. Determination at claim time based on project complexity.

**Three-Trades Rule on Tile Claims (almost always met):**
1. Roofing contractor (tile removal, installation, underlayment)
2. Sheet metal / copper fabrication (custom valleys, flashings)
3. Structural engineering (permit-required on historic structures or major decking)
4. Additional: HVAC relocation, chimney masonry, electrical

**Documentation:**
- Include O&P on every initial estimate — do not wait to be asked
- List all trades in scope narrative so three-trades basis is self-evident
- If denied, respond with written demand citing majority "reasonably likely" rule and trade count
- Request carrier's basis for denial in writing
- **Texas:** Carriers refusing O&P without legitimate basis subject to Prompt Payment Act penalties (18% annual interest + attorney fees)

> Document coordination costs explicitly: subcontractor meetings, scheduling conflicts, permit coordination, inspection attendance, punch-list management. Transforms abstract "O&P is owed" into specific "here is actual coordination work performed."

## Key Legal Cases & Precedents

### Schnell v. State Farm Lloyds — 5th Circuit 2024 (No. 22-10662)

**Most important recent case for specialty tile O&L claims in Texas.**

**Key Holding:** A building code official's written incompatibility determination creates a genuine issue of material fact as to whether partial tile repair was permissible under code. Carrier's denial of O&L without resolving this factual dispute was improper. Remanded to district court for trial.

**How to Use:**
- When carrier denies O&L despite code official's incompatibility letter: cite Schnell as authority that the question is triable — carrier cannot decide unilaterally
- When carrier argues determination was "conditional": Schnell addresses directly — conditional vs. absolute is a fact question for trial, not for the adjuster
- In appraisal: include Schnell citation in formal demand letter to signal legal sophistication

**What It Did NOT Decide:** Does not automatically trigger O&L from every compatibility letter — only holds that the letter raises a material fact question.

### Texas Prompt Payment Act — Insurance Code §542

- Carriers must acknowledge claims within 15 days
- Accept or deny within 15 business days of receiving all items
- Pay within 5 business days of acceptance
- **Failure triggers 18% annual interest + attorney's fees**

Track every deadline. Send certified mail demand letters when missed. Note interest accumulation in correspondence.

### Majority Rule on O&P

Multiple state and federal court decisions confirm O&P is owed when a GC is "reasonably likely" to be needed on a multi-trade project — even if one has not yet been contracted.

If carrier's position is "O&P is only owed after you hire a GC," their position is contrary to the majority rule.

## Specialty Claims Workflow — 9 Stages

| Stage | Key Actions | Common Failure Points |
|-------|------------|---------------------|
| **1. FNOL & Qualification** | Confirm storm date + hail size, order HailTrace, confirm O&L coverage amount on policy, confirm tile type | Failing to confirm O&L coverage before inspection |
| **2. Field Inspection** | Full photo protocol, mark test squares, complete C.A.R. assessment, note code issues, collect product markings | Inadequate test squares, missing attic inspection, no product markings |
| **3. Material Research** | Contact manufacturer for written compatibility/availability, document lead times, pull local codes | Verbal confirmation instead of written letter; not on letterhead |
| **4. Initial Estimate** | Build full estimate with ALL supplement items, include O&L in separate group, include O&P, write F9 notes | Missing supplement items; inadequate F9 notes; no supporting docs |
| **5. Adjuster Meeting** | Walk test squares together, present manufacturer letters + code analysis, document agreed/disputed items same day | No written follow-up after meeting |
| **6. Supplement Rounds** | Submit within 5 business days, document each disputed item with code/spec/standard | Accepting carrier's estimate without supplement |
| **7. O&L Activation** | If 25% rule: get permit conditions in writing. If incompatibility: get code official determination. Submit formal O&L demand letter | Burying O&L in Xactimate notes instead of formal demand letter |
| **8. Appraisal / Escalation** | Invoke appraisal clause after 2 supplement rounds, engage preferred appraiser, provide complete documentation | Missing appraisal demand deadline; incomplete documentation |
| **9. Final Payment** | Verify all approved items, confirm depreciation release schedule, confirm O&L separate from main coverage | Forgetting depreciation release; not tracking O&L as separate payment |

## Standard vs. Specialty Claims Comparison

| Factor | Standard Shingle | Specialty Tile/Slate |
|--------|-----------------|---------------------|
| Primary dispute | Price per square | Repair vs. replacement scope |
| Material availability | Off-the-shelf | May be discontinued/incompatible |

| Engineer reports | Uncommon | Routine — both sides use them |

| Appraisal/litigation rate | Low | High — carriers fight these |
