## ROOFING RULES — ALWAYS INCLUDE ON EVERY ROOFING ESTIMATE

### Mandatory on Every Roof Estimate
- **8 roofer hours** ($194 x 8 = $1,552.00)
- **Equipment operator** ($450.00)
- **Dumpster load** ($850.00)
- **Laminated roofs** → use "R&R Hip/Ridge cap - Standard profile" ($11.97/LF)
- **3-Tab roofs** → use "R&R Hip/Ridge cap - cut from 3 tab" ($10.76/LF)
- **Multiple layers:** add the "Add. layer" line item for each extra layer beyond the first
- **Felt:** 15 lb. at $48.12/SQ (note: 94 Theron used 30 lb. at $55.63 — use 15 lb. going forward per updated pricing)

### Steep Roof Charges — ALWAYS Include When Pitch ≥ 7/12
Steep charges apply to the square footage of EACH facet at that pitch. Both removal AND install charges apply.
- **7/12 to 9/12 pitch:** Remove $18.00/SQ + Install $64.07/SQ = $82.07/SQ total
- **10/12 to 12/12 pitch:** Remove $28.29/SQ + Install $100.73/SQ = $129.02/SQ total
- **Greater than 12/12 pitch:** Remove $36.24/SQ + Install $131.43/SQ = $167.67/SQ total
- Calculate steep charge SQ from EagleView: sum the area (SF) of all facets at that pitch tier, convert to SQ
- If roof has mixed pitches, apply the correct steep charge tier to each group of facets separately
- **NEVER omit steep charges when EagleView shows facets at 7/12 or steeper**

### High Roof Charge — ALWAYS Include When 2+ Stories
Both removal AND install charges apply to the FULL roof area.
- **Remove:** $7.31/SQ on total roof area
- **Install:** $30.44/SQ on total roof area
- Check EagleView "Number of Stories" field — if ≥ 2, include high roof charges
- **NEVER omit high roof charges when property is 2+ stories**

### Gable Cornice Returns — Include When Present
A gable cornice return is the decorative trim piece where the roof overhang wraps around the gable end and "returns" back to the wall — creating a boxed-in corner at the bottom of each gable. They are common on colonial, cape cod, and craftsman-style homes. Each gable end with a cornice return counts as 1 EA (some gables have returns on both sides = 2 EA per gable).
- **1 story:** Laminated $136.32/EA | 3-Tab $135.18/EA
- **2 stories or greater:** Laminated $165.29/EA | 3-Tab $164.15/EA
- Count from inspection photos or EagleView — look at each gable end
- If property has hip roof with no gable ends → no cornice returns
- **Include cornice returns whenever the roof style has gable ends with boxed returns visible in photos or EagleView**

## SIDING RULES — ALWAYS INCLUDE ON EVERY SIDING ESTIMATE

### Mandatory on Every Siding Estimate
- **Siding - Labor Minimum** ($519.46/EA) — ALWAYS include 1 EA on every siding estimate
- **House wrap** ($0.64/SF) — CODE REQUIRED per RCNYS R703.2. NEVER generate a siding estimate without this line item.
- **Scaffolding** ($1,405.00/WK) — ALWAYS include 1 week on every siding estimate
- **Wall flashing** at all openings — CODE REQUIRED per RCNYS R703.4
- **Corner rule:** House wrap must wrap around outside corners (R703.1/R703.2). If carrier approves only 1 elevation, this forces full replacement.

### Siding Type Identification — CRITICAL
- **ALWAYS confirm whether siding is vinyl or aluminum** before selecting line items. These are different materials with different prices.
- **Vinyl siding** → R&R Siding - vinyl ($7.55/SF), or High grade ($7.83), Premium grade ($8.21), insulated ($11.45), board & batten ($8.94)
- **Aluminum siding** → R&R Aluminum siding .024" ($12.47/SF)
- **Cedar shingle siding** → R&R Siding - cedar shingle ($18.13/SF)
- **Detach & reset** ($4.22/SF) — use when siding is being temporarily removed and re-installed (not replaced)

### Insulation Board
- **Fanfold foam insulation board** ($1.23/SF) — include when we confirm there is insulation board under the siding pre-loss. Do NOT auto-include — only when instructed.
